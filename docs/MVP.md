## 一、产品概述

**产品名称：** ID Photo Production（证件照在线制作工具）

**一句话定位：** 无需下载软件，上传照片即可在线生成符合标准的证件照，并支持打印排版下载。

**目标用户：** 需要制作身份证、护照、签证、简历等证件照的普通用户。

**核心价值：**
- 操作简单，3步完成（上传 → 选规格 → 下载）
- 无需注册，无需存储，保护隐私
- AI 智能抠图，背景替换精准
- 支持标准尺寸与打印排版，可直接送冲印店

## 四、技术架构

### 技术选型

| 层级 | 技术 |
|------|------|
| 前端框架 | Next.js 14（App Router）|
| UI 样式 | Tailwind CSS |
| 语言 | TypeScript |
| 图像处理 | Sharp.js（服务端，内存操作）|
| 背景去除 | Remove.bg API |
| 部署 | Vercel（前端）+ Railway（API）|

### 架构原则

- **无状态**：所有图片处理在内存中完成，不写入磁盘，不存数据库
- **隐私优先**：图片不落盘，处理完即销毁
- **前后端分离**：Next.js API Routes 承担服务端逻辑

### API 路由设计

| 路由 | 方法 | 功能 |
|------|------|------|
| /api/remove-bg | POST | 接收图片，调用 Remove.bg，返回去背 PNG |
| /api/process | POST | 接收去背图 + 背景色 + 规格，输出合成 JPG |
| /api/layout | POST | 接收处理后图片，输出 A4 排版 JPG |

## 五、用户流程

```
用户打开网站
    ↓
上传照片（拖拽 or 点击选择）
    ↓
系统自动调用 Remove.bg 去背（loading 动画）
    ↓
用户选择背景颜色（白/蓝/红）
    ↓
用户选择证件照规格（1寸/2寸/身份证/护照/签证）
    ↓
实时预览（左：原图，右：处理结果）
    ↓
点击「下载单张」或「下载A4排版」
    ↓
完成
```
